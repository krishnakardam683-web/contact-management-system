# Contact Management System (Java)

A simple **Console-based Contact Management System** developed in Java. This application allows users to manage their contacts efficiently through a menu-driven interface.

## Features

* Add New Contact
* View All Contacts
* Search Contact by Name
* Update Contact Details
* Delete Contact
* Exit Application

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Project Structure

```
ContactManagementSystem/
│── Contact.java
│── ContactManager.java
│── Main.java
```

## How to Run

1. Clone this repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or VS Code).
3. Compile and run the `Main.java` file.
4. Use the menu to manage contacts.
